User Manual

Movie App – User Manual

Welcome to the Movie App! This guide explains how to use all major features, including logging in, browsing movies, viewing details, managing ratings, favorites, and watchlists, and using your profile.

Login

Logging In

  1. Open the application.
  2. Enter your email and password.
  3. Tap Login to continue.

Registering a New Account

  • New users must register through TMDB.
  • Tap Register with TMDB to create an account.

Forgot Password

  • Select Forgot Password to be redirected to TMDB’s password recovery page.
